MRMG Interest Rate Python documentation¶

Installation [1]¶

  • General installation for Python and pip
  • General installation for git for MRMG git server
  • Package source code can be found from SunTrust internal server

Dependencies¶

  • numpy, matplotlib, bokeh, scipy are already installed by anaconda
  • mrmg (by pip install mrmg)
  • quantlib_python (by pip install quantlib_python)

Summary on the package¶

  • Yield term structure construction
  • Short rate modeling (calibration / simulation)
