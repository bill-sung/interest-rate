Hull-White one factor model class

Examples

>>> curve = FlatForward(ql.Date(19, 2, 2002), 0.04875825, daycount=DayCount.Act365)
>>> hw1f = HullWhite1F(curve)
>>> hw1f.calibrate('''calib_hw1f_euribor.json''')
>>> print(hw1f.param)
{'alpha': 0.04641697539064844, 'sigma': 0.005799411064368165}
>>> hw1f.simulate(ql.Period('1Y'), Freq.Quarterly, 1, random_seed=1000, is_plot=False)
2016-05-11 16:22:14 INFO Random seed is set as 1000
([0.0, 0.25, 0.5, 0.75, 1.0],
array([[ 0.04875825,  0.04567013,  0.04755594,  0.04992099,  0.05110811]]),
array([[ 1.        ,  0.98826584,  0.97701863,  0.96509589,  0.95302714]]))
